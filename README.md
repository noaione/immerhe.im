# immerhe.im

Hometown of [Cecilia Immergreen](https://www.youtube.com/@holoen_ceciliaimmergreen)

Built with [Bun](https://bun.sh/)

## Installation
1. Install [Bun](https://bun.sh/)
2. Run `bun install`
3. Run `bun generate`
4. Host the `dist` folder

Or, if you want dev server

1. Run with `bun dev`
